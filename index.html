API Endpoints

GET /health

Check if the bot is running and healthy.

POST /api/guardians/apply

Apply to become a Guardian. Body: { "discordId": "string", "wallet": "string", "note": "string" }

GET /api/guardians/list?status=pending

List guardians by status (requires admin auth).

POST /api/guardians/approve

Approve a guardian application (requires admin auth). Body: { "discordId": "string" }

POST /api/guardians/reject

Reject a guardian application (requires admin auth). Body: { "discordId": "string", "reason": "string" }

GET /api/roles

List all Discord roles (requires admin auth).

POST /api/roles/assign

Assign a role to a user (requires admin auth). Body: { "userId": "string", "roleName": "string" }

POST /webhooks/donation

Post a donation event (requires webhook signature).

Discord Commands

  • !ping - Check bot latency
  • !verify <wallet> - Verify wallet ownership of HealthID NFT
  • !impact - Stream live donation events
  • !linkwallet <wallet> - Link wallet (placeholder)
  • !assignrole - Assign roles (placeholder)
  • !help - Show available commands
